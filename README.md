# sentrik community

Welcome to the sentrik community hub! This repo hosts **Discussions**, **bug reports**, and **feature requests** for [sentrik](https://sentrik.dev) — the governance runtime for AI-generated code.

## Getting help

- **[Discussions](../../discussions)** — Ask questions, share tips, show what you've built
- **[Issues](../../issues/new/choose)** — Report bugs or request features
- **[Documentation](https://maxgerhardson.github.io/sentrik-docs)** — Installation, configuration, CLI reference, API docs

## Quick links

- **Install:** `npm install -g sentrik`
- **Scan:** `sentrik scan`
- **Gate:** `sentrik gate`
- **Dashboard:** `sentrik serve`
